# Merve Cafe — Personel Yönetim Release Dosyaları

Bu repo **sadece** Personel Yönetim uygulamasının otomatik güncelleme
sistemi için APK/EXE release dosyalarını barındırır.

Kaynak kod ve personel verileri burada **yoktur** — ana proje
(`personel_yonetim`) private bir repoda tutulur. Uygulama içindeki
güncelleme kontrolü (`update_service.dart`) sadece bu reponun
Releases sayfasını okur.
